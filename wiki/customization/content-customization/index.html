Content Customization

You can easily customize the content of your website by modifying the following directories:

Pages

The content displayed on your website pages can be found in the pages directory. Simply open the respective files to edit the text and layout as needed.

<template>
  <AppHeader title="Staff" />

  <Container>
    <!-- List of available colors can be found here: https://tailwindcss.com/docs/colors. -->

    <StaffGroup title="Management">
      <StaffMember name="Notch" rank="Owner" color="bg-red-700" />
      <StaffMember name="jeb_" rank="Lead developer" color="bg-yellow-700" />
    </StaffGroup>

    <StaffGroup title="Moderators">
      <StaffMember name="Rushboe" rank="Moderator" color="bg-blue-700" />
      <StaffMember name="Steve" rank="Moderator" color="bg-blue-700" />
      <StaffMember name="C418" rank="Trial moderator" color="bg-cyan-700" />
    </StaffGroup>
  </Container>
</template>

<script setup lang="ts">
useHead({
  title: 'Staff',
});
</script>

Navigation

Navigation links like the header menu can be configured in the app/app.config.ts file. You can edit the title, URL, icon, colors, and whether the link opens in a new tab. Example:

navigationLinks: [
  { title: 'Rules', to: '/rules', icon: 'fa-solid fa-scroll', colors: 'bg-emerald-800 hover:bg-emerald-900 border-emerald-950', external: false, blank: false },
  { title: 'Wiki', to: '/wiki', icon: 'fa-solid fa-book', colors: 'bg-purple-800 hover:bg-purple-900 border-purple-950', external: false, blank: false },
  { title: 'Store', to: '/store', icon: 'fa-solid fa-cart-shopping', colors: 'bg-amber-800 hover:bg-amber-900 border-amber-950', external: false, blank: false },
  { title: 'Voting', to: '/voting', icon: 'fa-solid fa-check-to-slot', colors: 'bg-rose-800 hover:bg-rose-900 border-rose-950', external: false, blank: false },
  { title: 'Staff', to: '/staff', icon: 'fa-solid fa-map', colors: 'bg-lime-800 hover:bg-lime-900 border-lime-950', external: false, blank: false },
]

Advanced Component Customization

For advanced users, components can be further customized by editing files in the app/components directory. This allows you to change layout, structure, or add new features, but these modifications are only recommended for users comfortable with Vue and TypeScript.
