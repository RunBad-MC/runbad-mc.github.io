Configuration

To set up your website, you'll need to configure a couple of key files.

Website Details

Edit the app/app.config.ts file to fill in the essential details about your website, such as the site title and description.

export default defineAppConfig({
  serverName: 'Your Minecraft Server',
  seoDescription: '',

  ...

});

Optional: Design Customization

You can customize certain design elements of your server site by editing the app/app.config.ts settings. This is optional and can be adjusted later.

For example, you can adjust:

...

logoSizeClasses: 'h-40 sm:h-48 max-w-64 md:max-w-80',
logoMarginClasses: 'mb-8',
backgroundDarkeningClass: 'bg-black/50',

...

Once you've made your changes, save the files. Your configurations will take effect the next time you run your development server or deploy the site.

If you need to set up the PHP API for the Voting Leaderboard, please refer to the Voting Leaderboard Setup Guide.
